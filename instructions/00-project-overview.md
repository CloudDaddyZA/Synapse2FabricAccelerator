# Project Overview
Create a Synapse to Microsoft Fabric Migration Accelerator that helps consultants understand a client's Synapse estate and prepare a practical migration path to Fabric.

## Primary Users
Data architects, cloud architects, Fabric consultants, migration consultants, delivery managers, technical SMEs.

## Main Deliverables
Synapse inventory (Excel + JSON), executive summary, technical audit, migration assessment, Synapse→Fabric mapping, wave plan, risk register, static HTML dashboard, Power BI datasets + model guide, GitHub Copilot optimization pack.

## Folder Structure
`.github/`, `instructions/`, `src/{agents,models,services,utils,exporters,templates}`, `config/`, `output/{discovery,inventory,assessment,migration,reports,dashboard,copilot_optimization_pack,logs}`, `powerbi/`, `tests/`, `docs/`.
